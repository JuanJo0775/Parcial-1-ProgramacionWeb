## 🚀 Quick Start

Para empezar rápido:

```bash
# 1️⃣ Instalar dependencias (solo primera vez)
npm install

# 2️⃣ Correr en desarrollo
npm run dev

# 3️⃣ Linting (antes de hacer push)
npm lint

# 4️⃣ Build para producción
npm run build
```

**Más información:** ver `README.md`

---

### 📦 Dependencias principales
- React 19.2.6
- TypeScript 6.0.2
- Vite 8.0.12
- Zustand 5.0.13

### ⚙️ Nota importante
La aplicación **siempre usa Mock API** (`VITE_USE_MOCK=true` en `.env`). No requiere backend real.
